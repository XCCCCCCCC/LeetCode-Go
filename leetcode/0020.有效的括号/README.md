# [有效的括号](https://leetcode.cn/problems/valid-parentheses/)

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

示例1：
```
输入：s = "()"
输出：true
```

示例2：
```
输入：s = "()[]{}"
输出：true
```

示例3：
```
输入：s = "(]"
输出：false
```

示例4：
```
输入：s = "([)]"
输出：false
```

示例5：
```
输入：s = "{[]}"
输出：true
```

## 解题思路
**思路1**：栈思想，设置括号匹配字典。遍历字符串，如果当前数组末尾重存在与之配对的括号，则出栈，进入下一次循环，否则进栈。循环结束判断数组是否为空即可。
