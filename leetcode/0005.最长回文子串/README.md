# [最长回文子串](https://leetcode.cn/problems/longest-palindromic-substring/)

给你一个字符串 s，找到 s 中最长的回文子串。

示例1：
```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

示例2：
```
输入：s = "cbbd"
输出："bb"
```

## 解题思路
**思路1**：暴力解法，列出所有子串判断，时间复杂度O(n^3)。
**思路2**：扩散中心法，两种场景判断，形如`bab`或`bb`，即偶数或奇数两种回文子串类型，如左右相等则继续向两侧扩散，直至不想等为止，时间复杂度O(n^2)
