# [最长公共前缀](https://leetcode.cn/problems/longest-common-prefix/)

编写一个函数来查找字符串数组中的最长公共前缀。
如果不存在公共前缀，返回空字符串 ""。

示例1：
```
输入：strs = ["flower","flow","flight"]
输出："fl"
```

示例2：
```
输入：strs = ["dog","racecar","car"]
输出：""
解释：输入不存在公共前缀。
```

## 解题思路
**思路1**：先将字符串数组按长度升序排序，最短字符串暂定为最长公共前缀，然后遍历比较字符串每一个位置是否相同，不同则退出循环并更新最长公共前缀。
